# Open Data AI Analytics: Аналіз екологічна та радіаційна обстановка в зоні розташування атомних електростанцій України

##Мета проєкту

Метою проєкту є аналіз екологічної та радіаційної обстановки в районах розташування атомних електростанцій України на основі відкритих державних даних. Проект спрямований на виявлення закономірностей у розподілі радіонуклідів, а також на перевірку дотримання встановлених норм безпеки.

## Джерело відкритих даних

Дані отримані з Єдиного державного веб-порталу відкритих даних:
[Екологічна та радіаційна обстановка в зоні розташування АЕС](https://data.gov.ua/dataset/4a9d3d56-bd95-4c3e-97e7-1cdc7bcbd445)

Дані містять інформацію про показники радіаційного контролю в зоні розташування атомних електростанцій.

## Питання / гіпотези для аналізу

1. Чи спостерігаються аномальні перевищення радіаційного фону в зоні розташування АЕС Гіпотеза: рівень радіації стабільний та не перевищує встановлених нормативних значень.

2. Чи існують значні коливання в об’ємах радіоактивних викидів (irg) залежно від кварталу? Гіпотеза: об’єми радіоактивних викидів можуть демонструвати сезонні коливання, пов’язані з опалювальним періодом або плановими ремонтними роботами

3. Чи існує статистично значущий взаємозв’язок між обсягами викидів cs_137_emission та co_60_emission? Гіпотеза: між показниками cs_137_emission та co_60_emission може існувати позитивна кореляція, що дозволяє припустити можливість прогнозування зміни одного радіонукліду на основі динаміки іншого.
